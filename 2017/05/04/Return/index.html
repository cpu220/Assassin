感慨

  • 本来是混coding，起始原因也是因为coding有个项目展示，这对于前端来说相当于免费的服务器。
  • 然而coding收费后，就懒得用了，加上加班、加班、加班的。1整年处于外网不能提交任何内容的状态，懒癌也犯了，毕竟是在受不了工作写一版，下班回来再写一版。打算再用的时候，coding对项目演示功能永久停用了。于是在懒癌又犯了一段时间后，还是打算迁移到github上好了。
  • 虽然写了很多脚手架，工具，包。然后却因为网络限制没有一个在外网存活着，不得不说是个悲哀。本着复习和学习的态度。后面会慢慢的迁移到github上，所以，立贴为证~！
